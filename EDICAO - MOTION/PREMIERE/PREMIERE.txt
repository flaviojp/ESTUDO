ATALHOS
----------------------------------------------------------------------
SETA + COMMAND -> MOVE A TRILHA DE VIDEO/AUDIO UM FREME (+ shift move mais)
L -> ADIANTAR O VÍDEO
J -> RETORNAR O VÍDEO
G -> AUDIO GAIN
C -> FERRAMENTA CORTE (GILETE), COM SHIFT CORTA TUDAS AS CAMADAS
N -> SERVE PARA MOVER O PONTO DE CORTE
CONTROL L -> SEPARAR OU JUNTAR A TRILA DE AUDIO DA TRILHA DE VDEO
TRANSIÇÃO DE ÁUDIO -> EFEITO / AUDIO TRANSITON / CONSTANTE POWER
SHIFT + D -> USA A TRANSIÇÃO PADRÃO
setar a transição como padrão (preferencia / timeline / definir um tempo padrão par)
TECLA ALT E SELECIONAR -> SELECIONA APENAS O ÁUDIO OU O VÍDEO
ALTERAR VALORES DE MANEIRA MIAS CONTROLADA, SEGURAR A TECLA CONTROL(lenta) OU SHIFIT(rápida), 
SHIT E -> ABILITA OU DESABILITA O ÁUDIO
COMAND U -> CRIA SUB-CLIPE
SHIFT I -> MOVE PARA O INICIO DA MARCACAO
SHIFT O -> MOVE PARA O FIM DA MARCACAO
VÍRGULA - ADICIONA O TRECHO EMPURRANDO
PONTO - ADICIONA O TRECHO SOBREESCREVENDO
ALT SETA PRA OU BAIXO -> LEVA OS ELEMENTOS PARA A CAMADE DE CIMA
SHIFT A -> FERRAMENTA QUE SELECIONA TUDO QUE EST DEPOIS DELA
SHIFIT CRASE -> DEIXA O PAINEL EM TELA CHEIA
SHIFIT -> SELECIONA VÁRIOS TRECHOS DA LINHA DO TEMPO
DEIXAR OS ÁUDIOS EM TRILHAS DIFERENTES
FERRAMENTA CANETA -> PARA CRIAR KEY FRAME NA TRILA DE ÁUDIO
SHIFT 7 -> EFEITOS
COLOR KEY -> USADO PARA CROMA KEY / ULTRA KEY É MELHOR
CRIAR UM COLO MATE NO BOTÃO DE CRIAÇÃO
EFEITO EDGE FEATHER -> USADO PARA DEIXAR OS CANTOS ARREDONDADOS
CROSS DISSOLVER -> EFEITO DE TRANSIÇÃO DE VÍDEO
ADD FRAME HOLD -> USADO PARA DEIXAR PARADO o FREME
EFEITO DE TRANSIÇÃO DE VDEO DE DIP TO WHITE -> PARA DEIXAR UMA TRANSIÇÃO BRANCA
TRANSIÇÃO DE VDEO WIPE DESLIZAR A TELA
EFEITO CROP PARA CORTAR PARTE DO VÍDEO
EFEITO GAUSSIAN BLUR
FAST COLOR CORRECTOR -> CORRETOR DE COR
SEQUENCE RENDER IN TO OUT -> USADO PARA DAR UM PRE RENDER NA TIMELINE DO VÍDEO TODO
